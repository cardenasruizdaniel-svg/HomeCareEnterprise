# Manuales por Rol — HomeCare Enterprise

Cada persona del equipo solo necesita leer **el manual de su propio rol** — no todo el sistema, solo lo que a usted le corresponde hacer.

| Manual | Para quién |
|--------|-----------|
| 01 — Administrador | Administrador, Director Médico |
| 02 — Coordinador | Coordinador |
| 03 — Médico | Médico |
| 04 — Enfermero | Enfermero / Enfermería |
| 05 — Terapeutas y Profesionales de Apoyo | Fisioterapeuta, Terapeuta Respiratorio, Psicólogo, Salud Ocupacional, Terapeuta, Nutricionista |
| 06 — Cuidador (solo app) | Cuidador |
| 07 — Administrativo | Administrativo, Auxiliar |
| 08 — Auditor | Auditor |

> Si su rol no aparece exactamente en esta lista (ej. Facturación, Inventario), consulte el **Manual de Funcionamiento** general — son roles especializados con acceso a un solo módulo específico.
