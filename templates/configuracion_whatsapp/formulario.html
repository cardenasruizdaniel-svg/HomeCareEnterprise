Antes de activar

Para que esto funcione de verdad, necesita tener (o crear) una cuenta de WhatsApp Business API con Meta:

  1. Cree una cuenta en business.facebook.com y active WhatsApp Business Platform.
  2. Obtenga el Token de acceso y el ID del número de teléfono desde el panel de desarrolladores de Meta.
  3. Registre esta URL como webhook en el panel de Meta:
    {{ request.url.scheme }}://{{ request.url.netloc }}/webhook/whatsapp
  4. Use el Token de verificación de abajo (ya generado automáticamente) cuando Meta se lo pida al registrar el webhook.

Mientras esto no esté configurado, los mensajes (órdenes médicas, confirmaciones de citas) se siguen registrando en el sistema pero no se envían de verdad — quedan en modo "simulado".
